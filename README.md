# gMeteoConnect
Cette application Flutter permet aux utilisateurs de consulter les prévisions météorologiques actuelles pour n’importe quelle ville dans le monde grâce à l’API OpenWeather.    L’utilisateur entre le nom d’une ville, appuie sur le bouton “Obtenir la météo”, et l’application récupère les données en temps réel via l’API.

Voir la branche master pour voir le projet : https://github.com/elegbede01/gMeteoConnect/tree/master
